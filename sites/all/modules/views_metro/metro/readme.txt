Installation
~~~~~~~~~~~~

Copy all the files (except this readme.txt file) in the root folder 
of your project. To change the properties of the page please use a 
text editor like Notepad to configure the file index.html.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1. To change the initial layout style go to line 15 and choose between
   horizontal or vertical in the class attribute: 
      <div class="metro-layout horizontal">
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
2. To change the title of the page modify the line 5 and line 17: 
      <title>Metro Flexible Layout</title> 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
3. The grid layout can have different box sizes and these can be 
   changed/added easily with classes. All you have to do is add a 
   widthX and heightX (X between 1-4).
   For example a simple box has the class="box"
      (defaults to width1, height1).
   A larger box may have class="box width2" - which means the width is 
      double the default and the height remains unchanged.
   A much bigger box may have class="box width2 height3" - width will 
      be double the default and the height will be tripple the default.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
4. There are plenty of icons available if you want to change them. Go to 
   the following websites to download more resources:
   http://www.iconarchive.com/show/windows-8-metro-icons-by-dakirby309.html
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Note
~~~~

If you want to add scrollbars to non Webkit browsers then go to line 32 of
the metro.css file and change the overflow attribute to auto:
   .metro-layout .content { height: 80%; overflow: auto; }
